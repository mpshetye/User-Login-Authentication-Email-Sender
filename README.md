## General info
A Blogging site to upload blogs with a sign up/login page for users, and a contact page
	
## Technologies
Project is created with:
* Bootstrap v5.0
* NodeJs v14.15.0
* npm v6.14.8
* Express ^4.17.1v
* Pug ^3.0.2v
* Mongoose ^5.12.9v
* Nodemom ^2.0.7v
* MongoDB
* Bcrypt ^5.0.1v
* Cookie-parser ^1.4.5v
* Dotenv ^9.0.2v
* jsonwebtoken ^8.5.1v
* Nodemailer  ^6.6.1v
* validator ^13.6.0v

	
## Setup
To run this project, install it using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
